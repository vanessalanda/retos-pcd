# Reto Semana 01 - Calculadora de Sumas

## Descripción

Programa en Python que procesa líneas de números separados por comas y calcula la suma de los valores válidos.

El programa limpia datos inválidos, maneja casos borde y trabaja únicamente con la parte entera de los números decimales.

---

## Características

- Suma números separados por comas
- Ignora caracteres inválidos
- Maneja líneas vacías
- Convierte valores inválidos en 0
- Trunca números decimales
- Maneja datos sucios como:
  - `3a4`
  - `5.7`
  - `abc`
  - `1,,2`

---

## Ejecución

Desde la terminal:

```bash
python main.py
